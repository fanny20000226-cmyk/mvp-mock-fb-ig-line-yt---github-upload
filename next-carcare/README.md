# CarCare NextJS Supabase Frontend

黑黃白多門店汽車內裝管理系統前端。

## 安裝

```bash
npm install
cp .env.example .env.local
```

將 Supabase anon key 填入 `.env.local`。

## 開發

```bash
npm run dev
```

## 檢查與打包

```bash
npm run lint
npm run typecheck
npm run build
```

## Vercel

專案 Root Directory 設定為：

```text
next-carcare
```
